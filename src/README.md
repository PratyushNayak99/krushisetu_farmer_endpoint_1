# AnnaData - Farmer App

**Blockchain for Agricultural Trust**

AnnaData is a comprehensive platform designed to empower small and marginal farmers in rural India, particularly in states like Odisha. This farmer-facing mobile application provides an intuitive interface for managing harvests, tracking produce through the supply chain, and receiving transparent payments.

## Features

### 🌾 Harvest Management
- Easy crop listing with visual selection
- Voice command integration for low-literacy users
- Multi-language support (Odia & English)
- Quality assessment and assayer requests

### 📊 Real-time Tracking
- Complete batch tracking from farm to market
- Visual timeline showing produce journey
- QR code generation for each batch
- Transparent status updates

### 💰 Financial Management
- Payment tracking and history
- Market price insights
- Earnings dashboard
- Bank account integration

### 🎯 User-Centric Design
- Large tap targets for ease of use
- Offline-first functionality
- Simple navigation
- Trust-building visual elements

## Target Users

Small and marginal farmers in rural India with:
- Low digital literacy
- Basic Android smartphones
- Intermittent internet access
- Local language preferences

## Technology

Built with modern web technologies:
- React with TypeScript
- Tailwind CSS for styling
- Responsive mobile-first design
- Progressive Web App capabilities
